# oawidget-cli

基于san的组件库cli工具

```
  Usage: oawidget [options]

  Options:
    -V, --version                   output the version number
    -c, --create                    创建项目脚手架
    -v, --view                      组件效果预览
    -t, --test                      组件规范检查
    -b, --build                     组件打包构建
    -p, --publish                   组件发布上线
    --config [key=value]            修改oawidget配置项
    --config list                   列出oawidget配置项
    -h, --help                      output usage information
```
